Take a look at what I've been working on!

React Dictionary Project

This dynamic dictionary app, built with React, HTML, CSS, and an API, allows users to search for word definitions, phonetics, and accompanying images. The app provides a clean, responsive interface and delivers fast, accurate results, making word exploration simple and engaging.

Oslo Travel Project

The Oslo Travel Project is a responsive web page designed to inspire and guide visitors to the vibrant city of Oslo. With a clean and engaging layout built using HTML, CSS, and Bootstrap, the site showcases Oslo’s top attractions and rich cultural experiences, making it easy for users to explore and plan their next trip.

React Weather App

A responsive weather app built with React, HTML, and CSS, offering real-time weather updates and a five-day forecast for any location. It provides key weather details like temperature, humidity, and wind speed, all through smooth API integration.

Travel Generator

This AI generated travel project generates must-visit places in entered destination for the user and also explains why you should visit these places. This app was built with HTML, CSS, AI API and JavaScript.
